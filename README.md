# Foxibar
Simulation of an ecosystem.

## Getting Started

### Prerequisites

Java
Eclipse
Maven

### Installing

Clone the project to a local repository.
I use ~/repo/Foxibar

Start eclipse and import:
 File -> Import...
 Maven -> Existing Maven Project
 Browse to your local clone -> Finish
